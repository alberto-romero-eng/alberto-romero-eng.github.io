(1) Adjust default workspace

Sugested: "~/ws-sts-4.x.x".

It might be useful to enable long paths/filenames in Windows environment, to avoid crashes.



(2) Adjust maven local repository path

(2.1) Adjust external 'settings.xml' file in 'Global' and 'User' fields:

Windows -> Preferences -> Maven -> User Settings

(2.2) Check correct path of '.m2' folder:

Windows -> Preferences -> Java -> Build Path -> Classpath Variables



(3) Adjust default JRE for workspace

(3.1) Add or edit available JREs:

Windows -> Preferences -> Java -> Installed JREs

Default eclipse JRE can be eliminated, leaving only customized ones.

(3.2) Verify default workspace JRE:

Windows -> Preferences -> Java -> Build Path -> Classpath Variables



(4) Install Darkest Dark Theme with DevStyle

(4.1) Install Darkest Dark Theme with DevStyle (icon-designer and code-together are not necessary):

Help -> Eclipse Marketplace -> Search

(4.2) Select 'Primary Colors' as icon-style:

Windows -> Preferences -> DevStyle

(4.3) If several workspaces are not necessary, use classic workspace selector with option 'do not ask again'



(5) Add lombok to ini file (do not add empty lines)

# lombok

-javaagent:lombok-1.18.24.jar



(6) Add icon scaling to ini file (do not add empty lines)

# GUI, icon scaling

-Dswt.enable.autoScale=true

-Dswt.autoScale=120

-Dswt.autoScale.method=nearest
